Theorem 1: In any graph, the number of nodes with an odd number of connections is always an even number.

Theorem 2: A graph is bipartite (can be split into two groups where no node connects to its own group) if and only if it has no odd-length cycles.

Theorem 3 (R(3,3)=6): For any graph G with exactly six vertices, either G contains a triangle, or its complement graph G' contains a triangle.

Theorem 4: If a graph is self-complementary (isomorphic to its own complement), then the number of vertices it has must be of the form 4n or 4n+1 for some integer n.
